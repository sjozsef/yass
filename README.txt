Smooth Scroll plugin for WordPress - Easy for beginners, advanced for power users.

== Description ==

YASS - Yet Another Smooth Scroll plugin for WordPress. Just install and feel the difference - your visitors will love it ;)

= Features: =

* Picasa-like smooth scrolling with mouse wheel and middle mouse button
* Supports keyboard (Arrow keys, PgUp/PgDown, Spacebar, Home/End)
* Full touchpad support
* Works with embedded content (PDF, Youtube video)
* Advanced, but easy to use settings
* Well optimized for speed

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/yass` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->YASS screen to configure the plugin
1. Enjoy the smoothest scroll you ever seen ;)
